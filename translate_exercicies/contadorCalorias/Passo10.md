# Passo 10

Seu menu selecionado precisa de opções para cada um dos elementos do conjunto de campos de alimentação e exercícios que você criou nas etapas anteriores. Use o elemento option para criar uma nova opção para cada conjunto de campos. O atributo value de cada opção deve ser o id do fieldset, e o texto de cada opção deve ser o texto da legenda.

Defina a opção Café da Manhã como a opção selecionada.
